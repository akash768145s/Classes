üéØ getElementById() - Find ONE element by ID
// Find the element with id="unique-element" const element = document.getElementById('unique-element'); // Check if we found it if (element) { element.style.background = 'yellow'; alert('Found: ' + element.textContent); }

Simple Explanation:

Finds ONE element that has a specific ID. IDs are unique - only one element can have the same ID.

üé® getElementsByClassName() - Find ALL elements by class
// Find ALL elements with class="common-class" const elements = document.getElementsByClassName('common-class'); // Loop through all found elements for (let i = 0; i < elements.length; i++) { elements[i].style.background = 'lightblue'; elements[i].textContent = 'Found me!'; }

Simple Explanation:

Finds ALL elements that have the same class name. Multiple elements can have the same class.

üè∑Ô∏è getElementsByTagName() - Find ALL elements by tag name
// Find ALL paragraph elements (p tags) const paragraphs = document.getElementsByTagName('p'); // Change all paragraphs for (let i = 0; i < paragraphs.length; i++) { paragraphs[i].style.background = 'pink'; paragraphs[i].style.padding = '10px'; }

Simple Explanation:

Finds ALL elements with the same HTML tag (like p, div, span, h1, etc.). Think "find all paragraphs" or "find all divs".

üéØ querySelector() - Find FIRST element using CSS
// Find FIRST element with class="common-class" const first = document.querySelector('.common-class'); // You can also find by ID or tag const byId = document.querySelector('#unique-element'); const byTag = document.querySelector('p'); // Only changes the FIRST one found first.style.background = 'orange';

Simple Explanation:

Finds the FIRST element that matches a CSS selector. Use . for class, # for ID, or just the tag name.

üéØ querySelectorAll() - Find ALL elements using CSS
// Find ALL elements with class="common-class" const allElements = document.querySelectorAll('.common-class'); // You can also find all paragraphs const allParagraphs = document.querySelectorAll('p'); // Loop through ALL found elements allElements.forEach((element, index) => { element.style.background = 'lightgreen'; element.textContent = `Found #${index + 1}`; });

Simple Explanation:

Finds ALL elements that match a CSS selector. Like querySelector but gets ALL matches, not just the first.
